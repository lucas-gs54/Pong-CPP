# 🏓 Ping! | Pong!
 
Recriação do Pong para aprender os fundamentos de C++ e desenvolvimento de jogos. Primeiro projeto utilizando a biblioteca Raylib, explorando conceitos como loop principal, renderização na tela, detecção de colisão e entrada do jogador.
 
## 🎮 Como jogar
 
| Jogador | Cima | Baixo |
|--------|------|-------|
| CPU (esquerda) | — | — |
| Jogador (direita) | `↑` | `↓` |
 
> A CPU acompanha a bolinha automaticamente. Tente fazer ela errar!
 
## ⚙️ Funcionalidades
 
- Raquete controlada por IA (CPU)
- Placar em tempo real
- Detecção de colisão entre bolinha e raquetes
- Efeito sonoro ao bater na raquete
- Bolinha reseta ao centro após cada ponto
## 🛠️ Tecnologias
 
- C++
- [Raylib](https://www.raylib.com/)
## ▶️ Como rodar
 
1. Clone o repositório
```bash
git clone https://github.com/lucas-gs54/Pong-CPP.git
```
 
2. Compile e rode seguindo as instruções do [Raylib CPP Starter Template](https://github.com/educ8s/Raylib-CPP-Starter-Template-for-VSCODE)
> Certifique-se de ter o arquivo `sfx/BarulhoBolinha.wav` na pasta do projeto para o som funcionar.
